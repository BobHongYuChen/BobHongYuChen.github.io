2020年度总结

引子

Dream as if you’ll live forever. Live as if you’ll die today. ——James Dean

从刚入行的新人，到看着新一届的实习生实习，新一届的应届生入职。很快，我也在OPPO一年半了

开始

过年的时候经历了一把疫情，当时最直观的感觉有两个，一个是每天的生活都充斥了许许多多的新闻，从武汉封城到雷神山火神山医院，另一个感觉就是每天起来假期就会被延长几天，看着假期不断增加还是挺开心的。

过去的一年有收获有遗憾，新的一年刚开始，就从收获开始说吧

收获

做饭

因为疫情原因，在深圳的生活的时候重拾了做饭这个已经被荒废许久的技能。一直到现在，周末的时候还是喜欢自己做饭吃，也算是自己减压的一种方式吧

成就感

从OPPO Relax开始到后面参与开发、维护的各个模块和项目都给我带来了一种特别的成就感，在微博刷到用户用了我参与开发的功能、在软件商店看着用户的评论，内心都有一种浓浓的成就感，虽然开发的过程中总有磕磕绊绊，也曾失落、气馁过，但是等到功能上线的时候，成就感总是能让我重新振作起来面对下一个问题

扩展爱好

读书&读书笔记

2020的一年中，我开始并坚持做读书笔记了。小时候每到寒假、暑假的时候，语文老师就会让写读书笔记，多则一两天一篇，少则一周一篇。从儿时开始做读书笔记就是一件无比痛苦的事，于是逐渐养成了看书但是不记书的习惯，但是随着书越读越多，突然发现自己没办法对每一本书的知识都做到建立索引，不同的书我对它的熟悉程度是不一致的，于是我开始尝试用XMIND来做自己的读书笔记，幸运的是，我对这个软件并不反感，甚至有点喜欢，也就一直坚持到了现在。

在深圳的日子，上班需要半小时的通勤时间，ipad拿着太大了，便逐渐开始用起了微信读书，在通勤或者其他的碎片的时间内让自己可以尽量多的汲取知识

相声

起初，是因为自己认床加上工作压力使然，我很成功的学会了失眠，有一次意外在B站刷到了郭老师和于老师的相声，于是喜欢上了相声

English

因为工作的原因会经常和国外的同事交流，开始强迫自己必须不依赖于翻译软件和他们交流，坚持了一年吧，现在已经可以比较熟练地用英语和他们交流了

Python

人生苦短，我用python。

最早的时候为了让自己填表没这么机械和繁琐，于是开始学习用python去帮助自己工作，2020年帮着组内的同事、设计师、产品写了挺多脚本，包括但不限于文件转换、数据处理可视化，可以说是一个很棒的收获了

关于身体、工作、生活

在2020年，加班是真的很多，项目也紧。年底体检报告出来之后，想了很久，突然意识到自己之前陷入了一个非常大的误区，那就是认为年轻的时候可以靠身体的底子让自己抗住工作，但是体检报告让我意识到，健康的身体需要持续的锻炼，虽然这是一个很简单的道理，但是一直以来我都没把这个简单的道理当回事，直到这次深刻体会了一把这个道理

遗憾

说到2020年的遗憾，其实也是关于在身体、工作和生活上的平衡，更确切的说其实就是三个字–拖延症，因为拖延症，自己想了很久的项目没有写、想做机械臂学了一部分之后也就没了下文、想系统的归纳和总结Android的知识、想多看看Android源代码，想了很多但是最后都拖延了，所以2021年一定要强制自己改掉这个问题
